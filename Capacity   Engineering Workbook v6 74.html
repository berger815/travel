⏳ Lead Times — Phase Configuration v6.74

Define independent lead time phases per MLI. Assign each department to exactly one phase per MLI. The model places full demand at Delivery Date − Phase Start Week for that department's capacity calculation.
⚠ Generate the workbook first (Setup tab) to configure lead time phases.
✅ All departments have valid phase assignments.
How it works: Each MLI has its own set of phases with independent start/end weeks. Each department is assigned to one phase per MLI. The model shifts that department's full demand by the phase Start Week before delivery. End week is visual only. Ship-date demand on the Demand tab is never modified — only the capacity calculation shifts. If a department has no phase assigned, demand appears at delivery date and an error is flagged.
